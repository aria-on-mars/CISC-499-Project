A platform for the analysis of DNA-transcription factor binding sites

This web server is for analyzing the binding sites between double-stranded DNA and transcription factors. A pipeline based on the command line (Li nux) has been proposed by Computational Biology and Biophysics Lab at Queen's University. The goal is to optimize the pipeline and run it as a graphical user interface. The web allows users to search and analyze the DNA-transcription factor binding sites from a user-provided PDB ID (the structure identifier used in the Protein Data Bank) or a PDB-formatted file. The results can be directly visualized or downloaded. This web server will help researchers to understand the molecular details of how the transcription factors recognize DNA and provide important clues for therapeutic targeting of oncogenic transcription factors.
